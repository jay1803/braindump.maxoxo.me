## Usage {#usage}

```swift
// 创建 StackView
var stackView = UIStackView()
// 设置间隔
stackView.spacing = 10
// 设定子视图的高度或宽度保持一致
stackView.distribution = .fillEqually

// 创建并添加内部视图
let textView = UITextView()
stackView.addArrangedSubview(textView)
```
